# Glossary — German care & medical terms in this assignment

You don't need to memorize these; your LLM knows them. This list exists so the German vocabulary is never the obstacle.

| Term | Meaning |
|---|---|
| **Arztbrief / Entlassungsbrief** | Hospital discharge letter, written to the GP (Hausarzt). The authoritative summary of a hospital stay. |
| **Epikrise** | The narrative "course and treatment" section of a discharge letter. Findings are often buried here. |
| **Entlassmedikation** | The discharge medication list — the prescription state at discharge. |
| **Procedere** | "Further steps / recommendations" section: follow-ups, monitoring instructions, GP tasks. |
| **Diagnosen / Nebendiagnosen** | Main and secondary diagnoses, usually with ICD-10 codes. |
| **Anamnese / Sozialanamnese** | Patient history / social history (living situation, family). Often contains non-actionable narrative. |
| **Hausarzt (HA)** | General practitioner; the typical addressee for follow-up tasks ("b. HA" = at the GP). |
| **Pflegegrad** | German care-level grading 1–5 (5 = highest need). |
| **Verlaufsdoku / Pflegebericht** | Nursing progress notes, written per shift (early/late/night). |
| **Trinkprotokoll** | Fluid-intake protocol: target amount (Soll) per day plus logged intake entries. |
| **Bedarfsmedikation (b.B. / n.B.)** | On-demand (PRN) medication, with a maximum daily dose. "b.B./n.B." = bei/nach Bedarf (as needed). |
| **Wirkstoff** | Active substance of a drug. German documents freely mix brand names and substance names — e.g. Aspirin vs. Acetylsalicylsäure, Voltaren vs. Diclofenac. |
| **Dekubitus** | Pressure ulcer, graded I–IV ("Grad/°"). |
| **Durchgangssyndrom** | Older German clinical term for a transient post-operative confusional state (delirium). |
| **Exsikkose** | Dehydration. |
| **Z.n.** | "Zustand nach" = status post / history of. |
| **AZ / EZ** | Allgemeinzustand / Ernährungszustand = general / nutritional condition ("red." = reduced). |
| **RR / HF / BZ / Krea / GFR / CRP / INR** | Blood pressure / heart rate / blood sugar / creatinine / kidney function / inflammation marker / coagulation value. |
| **Dosierung "1-0-1-0"** | Intake scheme: morning–noon–evening–night. |
| **VW / Rö-Ko. / Labor-Ko.** | Verbandwechsel (dressing change) / X-ray check / lab check. |
| **Anlage** | Attachment/appendix of a letter ("siehe Anlage 2" = see attachment 2). |
| **NYHA I-IV** | Heart-failure severity classification (IV = worst). |
| **EF** | Ejection fraction, a heart-function measure (normal > 50 %). |
| **Überleitung** | Care transition between hospital and nursing home — the process this assignment is about. |
